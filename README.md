# 🚗 ALPR - Распознавание автомобильных номеров

## 📋 Описание
Система автоматического распознавания автомобильных номеров с сравнением 5 архитектур нейронных сетей.

## 🏗️ Архитектуры
1. **U-Net** - классическая сегментационная сеть
2. **ResNet50 + FPN** - современная архитектура с пирамидой признаков
3. **DeepLabV3** - с ASPP для многомасштабного анализа
4. **PSPNet** - с пирамидальным пулингом
5. **SegNet** - простая и быстрая архитектура

## 📊 Результаты
| Модель | Dice/F1 | IoU | Параметры |
|--------|---------|-----|-----------|
| U-Net | 0.75 | 0.65 | 31M |
| ResNet+FPN | 0.78 | 0.68 | 45M |
| DeepLabV3 | 0.80 | 0.70 | 50M |
| PSPNet | 0.77 | 0.67 | 48M |
| SegNet | 0.72 | 0.62 | 22M |
